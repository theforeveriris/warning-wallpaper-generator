# 保密壁纸生成器

一个功能强大的保密警告壁纸生成工具，用于创建带有自定义保密信息的桌面壁纸。

## 功能特性

- 🎨 **自定义背景**：支持设置背景颜色和壁纸比例
- 📝 **中心文字**：自定义文字内容、颜色、字体、大小、位置和对齐方式
- 🖼️ **标志设置**：支持使用预设标志或上传自定义标志
- 🔍 **落款设置**：添加自定义落款文字，支持设置字体、颜色和位置
- 💾 **多格式下载**：支持 PNG 和 JPG 格式下载
- 📱 **响应式设计**：适配桌面端和移动端
- 🔧 **配置管理**：支持导出和导入配置
- 🎯 **实时预览**：配置变更实时反映在预览效果中

## 技术栈

- **前端框架**：Vue 3 + TypeScript
- **构建工具**：Vite
- **样式**：原生 CSS + 响应式设计
- **Canvas API**：用于壁纸渲染

## 快速开始

### 安装依赖

```bash
npm install
```

### 开发模式

```bash
npm run dev
```

项目将在本地启动，默认地址为 `http://localhost:5173`。

### 构建项目

```bash
npm run build
```

构建后的文件将生成在 `dist` 目录中。

### 预览构建结果

```bash
npm run preview
```

## 使用说明

### 基本使用流程

1. **选择标签页**：在顶部标签栏选择需要设置的选项（总编、中心文字、标志、落款）
2. **调整配置**：根据需要修改各项配置参数
3. **实时预览**：右侧预览区会实时显示效果
4. **下载壁纸**：在预览区下方选择下载格式并点击下载按钮

### 配置选项说明

#### 总编
- **背景颜色**：设置壁纸背景颜色
- **壁纸比例**：选择预设比例或自定义尺寸
- **配置管理**：导出/导入配置、重置配置
- **自定义远程字体**：添加自定义字体

#### 中心文字
- **文字内容**：设置中心警告文字
- **文字颜色**：设置文字颜色
- **字体**：选择字体类型
- **字体大小**：调整文字大小
- **行高**：设置文字行高
- **水平位置**：调整文字水平位置
- **垂直位置**：调整文字垂直位置
- **文本对齐**：设置文字对齐方式

#### 标志
- **预设标志**：选择预设标志
- **上传标志**：上传自定义标志图片
- **标志大小**：调整标志大小
- **水平位置**：调整标志水平位置
- **垂直位置**：调整标志垂直位置

#### 落款
- **落款内容**：设置落款文字
- **字体**：选择落款字体
- **文字颜色**：设置落款颜色
- **字体大小**：调整落款大小
- **水平位置**：调整落款水平位置
- **垂直位置**：调整落款垂直位置
- **文本对齐**：设置落款对齐方式

## 部署指南

### GitHub Pages 部署

1. **构建项目**：
   ```bash
   npm run build
   ```

2. **创建 docs 目录**：
   ```bash
   mkdir docs
   cp -r dist/* docs/
   ```

3. **提交到 GitHub**：
   ```bash
   git add docs
   git commit -m "Add build files for GitHub Pages"
   git push
   ```

4. **配置 GitHub Pages**：
   - 进入 GitHub 仓库设置
   - 选择 "Pages" 选项
   - 选择 "Deploy from a branch"
   - 选择 "main" 分支和 "/docs" 目录
   - 点击 "Save"

### 其他部署方式

构建后的 `dist` 目录可以部署到任何静态网站托管服务，例如：
- Vercel
- Netlify
- AWS S3 + CloudFront
- 阿里云 OSS + CDN
